# Getting Started

Welcome to JournalClub! This tool uses AI to summarize and narrate medical research papers, making them easy to consume during your commute or workout.

## Quick Start

1. **Upload a Paper** - Click the upload box and select a PDF of a medical journal article (max file size: 10MB)
2. **Choose Your Action**:
   - **Summarize** - Get an AI-generated summary with key points, methodology, and conclusions
   - **Listen** - Generate an audio narration optimized for listening
3. **Create Topics** - Group up to 5 related papers into a topic for a cohesive listening experience

## Features

- 📄 PDF parsing and text extraction
- 🤖 AI-powered summarization
- 🎧 Text-to-speech using Microsoft Edge neural voices (no API key required)
- 📻 RSS podcast feed for your favorite podcast app
- ⏰ 24-hour retention (perfect for daily use)
